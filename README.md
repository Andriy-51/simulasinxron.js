Simulasinxron - Coursework: Asynchronous Request Processing Simulator

Опис
------
Цей проєкт демонструє імітацію асинхронної обробки запитів на сервері в Node.js. Він об'єднує кілька навчальних модулів у єдиний демонстраційний сценарій.

Що реалізовано
- Генератори / Ітератори (round-robin)
- Черга з пріоритетами (BiDirectionalPriorityQueue)
- Асинхронна обробка (asyncMap)
- Кешування (memoize з LRU/TTL варіантами)
- Обробка великих даних (chunking, streams)
- Event-driven комунікація (ReactiveEmitter / Observable)
- Просте логування та моніторинг

Файли
- `demaTask1.js` — головний демонстраційний сценарій з меню
- `src/` — реалізації модулів по лабораторних
- `docs/COURSEWORK.md` — коротка методичка для презентації

Швидкий запуск
```bash
cd simulasinxron.js
npm install
npm run demo
```

Якщо запускаєш саме з Windows PowerShell і `npm start` блокується політикою виконання, використовуй:
```bash
npm.cmd run demo
```

Для простого показу викладачу можна запустити файл `run-demo.cmd` у корені проєкту через Провідник Windows. Після запуску відкриється меню в консолі. Можна:
- обрати одну секцію за номером;
- запустити весь демо-прогін;
- запустити кастомний прогін і ввести свої запити або елементи черги.

Поради для захисту
- Демонструй секції по порядку (генератори → черга → async → кеш → великі дані → реактивність → логування).
- Показуй короткі фрагменти коду з `src/` для пояснення підходів (наприклад, `memoize` та `BiDirectionalPriorityQueue`).
