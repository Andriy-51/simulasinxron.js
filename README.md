Simulasinxron - Coursework: Asynchronous Request Processing Simulator

Опис
------
Цей проєкт демонструє імітацію асинхронної обробки запитів на сервері в Node.js. Він об'єднує кілька навчальних модулів у єдиний демонстраційний сценарій.

Що реалізовано
- Генератори / Ітератори (round-robin)
- Черга з пріоритетами (BiDirectionalPriorityQueue)
- Асинхронна обробка (asyncMap)
- Кешування (memoize з LRU/TTL варіантами)
- Обробка великих даних (chunking, streams)
- Event-driven комунікація (ReactiveEmitter / Observable)
- Просте логування та моніторинг
